# Sudoku Validity Checker

A console-based Sudoku Validity Checker built in C++ using Object-Oriented Programming concepts.

## 👥 Developed By
- Amna Abdul Qadir
- Huda Naqvi

## 📌 About
This program loads a pre-defined 9x9 Sudoku board and allows the user to check 
whether a number placed at a given position is a valid move or not.

## ⚙️ Features
- Displays the current Sudoku board
- Validates moves based on Sudoku rules (row, column, and 3x3 grid)
- Exception handling for invalid inputs
- OOP-based design with encapsulated Sudoku class

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Clone the repository
2. Compile: `g++ DS_Project_Code.cpp -o sudoku`
3. Run: `./sudoku`
